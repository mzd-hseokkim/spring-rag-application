# Phase 14 — UX 개선 + 디자인 리뉴얼

컴포넌트를 미려하게 다듬고, 브랜드 컬러를 적용하여 전체적인 디자인 품질을 높인다.

---

## 1. 브랜드 컬러 적용

### 컬러 팔레트
- **Primary**: #011936 (딥 네이비)
- **Secondary**: #465362 (슬레이트 그레이)
- **Accent/Error**: #ED254E (비비드 레드)

### 적용 범위
- Tailwind CSS 변수 / shadcn 테마 변수 수정 (index.css)
- 버튼, 배지, 토글, 링크 등 인터랙티브 요소
- 사이드바 배경/활성 상태
- 로그인/회원가입 페이지 헤더

## 2. 컴포넌트 미려화

### 2-1. 채팅 영역
- 메시지 버블 디자인 개선 (유저/AI 구분 강화)
- 에이전트 스텝 표시 애니메이션
- 빈 상태(Empty state) 일러스트/안내 문구
- 스트리밍 중 타이핑 인디케이터

### 2-2. 사이드바
- 대화 목록 hover/active 효과 개선
- 문서 상태 아이콘 + 프로그레스 인디케이터
- 태그/컬렉션 칩 색상 다양화

### 2-3. 관리자 페이지
- 테이블 디자인 통일 (스트라이프, 호버 효과)
- 대시보드 카드 그림자/테두리 개선
- 차트 색상 브랜드 컬러 연동

### 2-4. 공통
- 페이지 전환 애니메이션
- 로딩 스켈레톤 (데이터 로드 중)
- 토스트 알림 스타일 브랜드 컬러 적용
- 반응형 개선 (최소 너비 대응)

## 3. 다크모드
- 브랜드 컬러 기반 다크모드 팔레트
- next-themes 활용 (이미 설치됨)
- 사이드바 하단에 테마 토글 버튼
