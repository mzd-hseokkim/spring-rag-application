## 매핑 규칙
1. 각 요구사항을 **하나의 목차 항목**에 배치 (가장 하위 레벨 항목에 배치)
2. 매핑 대상이 없는 요구사항은 가장 유사한 항목에 배치
3. 하나의 목차 항목에 여러 요구사항이 배치될 수 있음
4. 모든 요구사항을 빠짐없이 매핑

## 출력 형식

키는 목차 key, 값은 해당 목차에 매핑된 요구사항 id 배열입니다.

  "1.1": ["REQ-01", "REQ-02"],
