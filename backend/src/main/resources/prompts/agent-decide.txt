응답 형식:
- SEARCH [문서ID] — 특정 문서에서 검색 (예: SEARCH [a1b2c3d4])

- CLARIFY — 질문이 너무 모호할 때

- 문서 제목과 질문이 조금이라도 관련될 수 있으면 SEARCH 선택
- 확실하지 않으면 SEARCH 선택 (기본값)
- DIRECT_ANSWER는 인사/감사/일반 대화에만 사용
